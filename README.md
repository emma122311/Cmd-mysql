# Cmd-mysql
### 在 cmd中 匯入 masql 中 步驟
```
 1.在電腦中的搜尋欄位上輸入 cmd 後按確定
 2.把要匯入的 masql 放入 C:/xampp/mysql/bin 中
 3.在 cmd 中輸入cd c:/——cd xampp —— cd mysql —— cd bin 
 4.例如：  C:/xampp/mysql/bin 下輸 入mysql -u root 資料庫名稱(所要匯入的 mysql ) < XXXXX.sql
 
 //政軍的步驟：
 //CREATE DATABASE www_admin
 //mysql -h localhost -uroot -p
 //mysql -h localhost -u root -p www_admin   < c:/www_admin.sql
```
